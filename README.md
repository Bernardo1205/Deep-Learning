### CV Creator Application

A user-friendly command-line application to help you create professional CVs (Curriculum Vitae).

#### Quick Start

```bash
python3 cv_creator.py
```

#### Features
- Interactive step-by-step CV creation
- Multiple output formats (JSON and formatted text)
- Comprehensive sections: Personal Info, Education, Experience, Skills, Languages, Certifications
- Preview before saving
- No external dependencies required
